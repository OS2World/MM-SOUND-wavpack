The Monkey's Audio Codec player for PM123
                             Version 0.00


はじめに
----------
Monkey's Audio Codec形式のファイルを
PM123で再生するためのプラグインです。


動作環境
-----------
PM123 version 1.30が正常に動作する環境
もしかすると、libcのランタイムが必要かも
しれません。(libc06r1.dll)


インストール方法
------------------
macplay.dllを適切なフォルダにコピーし
PM123のプラグインの設定でmacplay.dllを
追加してください。


アンインストール方法
---------------------
iniファイルなどは使ってませんので、
PM123でプラグインの設定を削除した後
macplay.dllを削除してください。
